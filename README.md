# 🎓 SmartEdu – School Management System

**SmartEdu** is a modern, lightweight, and scalable **school management system** designed to simplify the operations of educational institutions.  
It includes role-based dashboards for **Admins**, **Teachers**, and **Students** — all powered by a Node.js + Express backend and a responsive HTML/CSS frontend.

---

## 🚀 Features

### 👩‍💼 Admin
- Manage students, teachers, and subjects
- View overall academic performance
- Approve and update results

### 👨‍🏫 Teacher
- Manage assigned subjects and classes
- Upload and update student results
- Communicate with admin or students

### 👩‍🎓 Student
- View personal dashboard and results
- Access performance reports
- Receive notifications from teachers/admin
